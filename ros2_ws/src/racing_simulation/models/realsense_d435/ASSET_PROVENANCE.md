# D435 mesh provenance

`meshes/d435.dae` was copied from the working legacy NUC simulation at:

```text
uav_test/src/PX4-Autopilot/Tools/simulation/gz/models/realsense_d435/meshes/d435.dae
```

The legacy directory did not contain a license or source reference for this
mesh. The file is retained at the project owner's request so the migrated model
matches the existing simulation visually. Before publishing this repository or
redistributing the mesh outside the team, identify its original source and
confirm that its license permits redistribution.
